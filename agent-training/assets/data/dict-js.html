1. 마법 지팡이 (요소 찾기)

  • document.getElementById('ID'): HTML에서 특별한 이름(id)을 가진 요소를 한 개 찾아냅니다.
  • document.querySelector('.class'): CSS 선택자 방식으로 요소를 찾아냅니다. (매우 강력!)
  • const 상자 = ...: 변하지 않는 값을 담는 상자(상수)를 만듭니다.
  • let 상자 = ...: 언제든 바뀔 수 있는 값을 담는 상자(변수)를 만듭니다.

2. 기초 마법 (내용과 디자인 바꾸기)

  • 요소.innerText = "내용": 화면에 보이는 글자를 바꿉니다.
  • 요소.innerHTML = "<b>내용</b>": HTML 태그까지 포함해서 내용을 바꿉니다.
  • 요소.style.속성 = "값": 디자인을 바꿉니다. (예: style.color = "red")
  • 요소.classList.add('이름'): 클래스를 추가해서 미리 만들어둔 스타일을 적용합니다.

3. 궁극기 마법 (함수와 이벤트)

  • function 이름() { }: 여러 마법을 하나로 묶어 '마법 주문'을 정의합니다.
  • 요소.onclick = 함수: 요소를 클릭했을 때 주문을 발동시킵니다.
  • 요소.addEventListener('이벤트', 함수): 더 세밀하게 이벤트를 감시합니다. (예: 'mouseover', 'keydown')
  • alert("메시지"): 팝업 알림창을 띄웁니다.

4. 데이터와 논리 마법

  • console.log(값): 개발자 도구 콘솔에 비밀 메시지를 출력합니다. (디버깅용)
  • if (조건) { } else { }: 상황에 따라 마법을 다르게 씁니다.
  • === / !==: 두 값이 같은지 혹은 다른지 비교합니다.
  • Array (배열): 여러 데이터를 [1, 2, 3] 처럼 순서대로 모아둡니다.
  • Object (객체): 데이터를 { name: "XYLO" } 처럼 이름표를 붙여 모아둡니다.

5. 외부 세계와 통신 (API)

  • fetch('URL'): 서버에서 실시간 데이터를 가져오는 통신 마법입니다.
  • .then(): 통신이 성공한 뒤에 할 일을 순서대로 정합니다.
  • JSON.parse(): 서버에서 온 암호 같은 데이터를 JS가 읽을 수 있게 바꿉니다.
