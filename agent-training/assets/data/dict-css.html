1. 텍스트 디자인 속성

  • color 글자 색상 지정
  • font-size 글자 크기 조절
  • font-weight 글자 두께 설정
  • font-weight="bold" 굵은 글씨 설정
  • font-weight="normal" 보통 글씨 설정
  • font-family 사용 글꼴 지정
  • text-align 텍스트 정렬 방식
  • text-align="center" 가운데 정렬
  • text-align="left" 왼쪽 정렬
  • text-align="right" 오른쪽 정렬
  • text-decoration 글자 꾸밈 (밑줄 등)
  • text-decoration="none" 장식 없음 (링크 밑줄 제거)
  • line-height 줄 간격 높이
  • letter-spacing 글자 사이 간격

2. 박스 모델 (Box Model)

  • width 가로 너비 지정
  • height 세로 높이 지정
  • padding 안쪽 여백 설정
  • margin 바깥 여백 설정
  • margin="0 auto" 블록 요소 가운데 정렬
  • border 테두리 종합 설정
  • border-style 테두리 선 종류
  • border-style="solid" 실선 테두리
  • border-style="dashed" 점선 테두리
  • border-radius 모서리 둥글게 조절
  • box-shadow 박스 그림자 효과
  • box-sizing 크기 계산 방식 설정
  • box-sizing="border-box" 여백/테두리 포함 크기 계산
  • overflow 넘침 영역 처리
  • overflow="hidden" 넘친 내용 숨기기
  • overflow="scroll" 항상 스크롤바 표시

3. 배경 디자인

  • background-color 배경 색상 지정
  • background-image 배경 이미지 삽입
  • background-size 배경 크기 조절
  • background-size="cover" 배경 꽉 채우기
  • background-size="contain" 비율 유지하며 맞추기
  • background-position 배경 이미지 위치
  • background-repeat 배경 반복 여부
  • opacity 전체 투명도 조절

4. 레이아웃 & 위치 (Layout)

  • display 표시 및 배치 방식
  • display="block" 한 줄 전체 차지
  • display="inline" 콘텐츠만큼만 차지
  • display="flex" 유연한 박스 레이아웃
  • display="grid" 격자무늬 레이아웃
  • display="none" 요소 완전히 숨김
  • position 위치 기준 설정
  • position="relative" 기존 위치 기준 이동
  • position="absolute" 부모 기준 절대 위치
  • position="fixed" 화면 기준 고정 위치
  • position="sticky" 스크롤 시 특정 지점 고정
  • top / bottom / left / right 방향별 이동 거리
  • z-index 요소 수직 쌓임 순서
  • float 요소 좌우 배치
  • visibility 가시성 여부 (공간 유지하며 숨김)

5. 플렉스 박스 (Flexbox)

  • flex-direction Flex 정렬 방향
  • flex-direction="row" 가로 방향 정렬 (기본)
  • flex-direction="column" 세로 방향 정렬
  • justify-content 메인축 정렬 방식
  • justify-content="center" 가운데 정렬
  • justify-content="space-between" 양끝 배치 및 사이 균등 간격
  • align-items 수직축 정렬 방식
  • flex-wrap Flex 줄바꿈 설정
  • gap 요소 사이 간격 설정

6. 애니메이션 & 효과

  • transition 상태 변화 지속 시간
  • transform 요소 회전/크기 변형
  • transform="scale(1.1)" 1.1배 확대
  • transform="rotate(45deg)" 45도 회전
  • cursor 마우스 커서 모양
  • cursor="pointer" 손가락 모양 (클릭 가능)
  • cursor="default" 기본 화살표 모양
  • animation 키프레임 애니메이션 적용
