1. 웹 문서 기본 뼈대 태그

  • <html> 문서 정의 태그: 웹 문서의 시작과 끝을 나타냅니다. 모든 내용은 이 안에 들어갑니다.
  • <head> 머리말 태그: 브라우저에게 문서의 정보를 알려줍니다. 화면에 직접 보이지 않습니다.
  • <meta> 메타 태그: 문서의 문자 인코딩 설정 등 부가 정보를 담습니다. (예: <meta charset="UTF-8">)
  • <title> 제목 태그: 브라우저 탭에 표시될 웹페이지의 제목을 정합니다. (예: <title>도로랜드</title>)
  • <body> 내용(본문) 태그: 실제로 화면에 보여질 모든 내용이 들어가는 공간입니다.

2. 제목 & 텍스트 꾸미기 태그

  • <h1> ~ <h6> 제목 태그: 글의 제목을 만듭니다. h1이 가장 크고 h6이 가장 작습니다.
  • <p> 단락 태그: 일반적인 문장을 작성할 때 사용하며, 위아래에 약간의 여백이 생깁니다.
  • <b> 굵은 글씨 (Bold): 글자를 단순히 굵게 만듭니다.
  • <strong> 강조 (Strong): 내용이 '중요함'을 의미하며 굵게 표시됩니다.
  • <i> 기울임꼴 (Italic): 글자를 단순히 기울여 씁니다.
  • <em> 강조 (Emphasis): 내용을 강조할 때 쓰며 기울임꼴로 표시됩니다.
  • <span> 범위 태그: 문장 안에서 특정 부분만 묶어 스타일을 줄 때 사용합니다.
  • <br> 줄바꿈 태그: 강제로 줄을 바꿉니다. 닫는 태그가 없습니다.

3. 목록 & 링크 태그

  • <ul> 순서 없는 목록: 점(bullet)으로 표시되는 목록을 만듭니다.
  • <ol> 순서 있는 목록: 숫자(1, 2, 3...)로 표시되는 목록을 만듭니다.
  • <li> 목록 항목: ul이나 ol 안에서 실제 항목을 나타낼 때 씁니다.
  • <a> 링크 태그: 다른 페이지나 파일로 연결합니다. (예: <a href="주소">텍스트</a>)

4. 멀티미디어 & 입력 태그

  • <img> 이미지 태그: 이미지를 화면에 넣습니다. (예: <img src="이미지경로" alt="설명">)
  • <input> 입력창 태그: 사용자에게 입력을 받습니다. type 속성에 따라 모양이 바뀝니다.
  • <button> 버튼 태그: 클릭할 수 있는 버튼을 만듭니다.
  • <textarea> 여러 줄 입력창: 긴 글을 입력받을 수 있는 박스를 만듭니다.
  • <label> 이름표 태그: 입력창에 이름을 붙여줍니다. 클릭 시 연결된 입력창으로 포커스가 갑니다.

5. 공간 나누기 & 구조 태그 (Layout)

  • <div> 박스 태그: 여러 요소들을 하나의 투명한 상자에 담아 묶어줍니다.
  • <header> 상단 영역: 페이지 상단의 로고나 메뉴 등이 들어가는 공간입니다.
  • <nav> 내비게이션: 메뉴 버튼들이 모여 있는 곳임을 알려줍니다.
  • <main> 주요 내용: 페이지의 핵심 콘텐츠가 들어가는 공간입니다.
  • <section> 섹션: 문서의 주제별 구역을 나눌 때 사용합니다.
  • <article> 독립적 글: 뉴스 기사나 블로그 포스트처럼 그 자체로 독립적인 내용을 담습니다.
  • <footer> 하단 영역: 페이지 하단의 저작권 정보, 제작자 정보 등이 들어갑니다.

6. 표(Table) 태그

  • <table> 표 정의: 표 전체를 감싸는 태그입니다.
  • <tr> 표의 행(Row): 가로 한 줄을 나타냅니다.
  • <th> 표의 제목 칸: 제목이 되는 칸이며 굵게 표시됩니다.
  • <td> 표의 내용 칸: 실제 데이터가 들어가는 일반적인 칸입니다.
