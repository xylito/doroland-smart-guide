1. 웹 문서 기본 뼈대 태그

  • <html> 문서 정의 태그
  • <head> 머리말 태그 (정보)
  • <meta> 메타 정보 설정
  • <title> 문서 제목 설정
  • <body> 본문 내용 구역
  • lang 문서 사용 언어 (ko, en)

2. 제목 & 텍스트 꾸미기 태그

  • <h1> ~ <h6> 제목 (크기별)
  • <p> 문단 태그
  • <strong> 강한 강조 (굵게)
  • <em> 의미 강조 (기울임)
  • <span> 인라인 범위 지정
  • <br> 줄바꿈 태그
  • <blockquote> 인용문 구역
  • <hr> 가로 구분선
  • <pre> 형식화된 텍스트 (여백유지)
  • <code> 코드 조각 표시

3. 목록 & 링크 태그

  • <ul> 순서 없는 목록
  • <ol> 순서 있는 목록
  • <li> 목록 아이템
  • <a> 하이퍼링크 태그
  • href 연결 주소 (URL)
  • target 창 열기 방식
  • target="_blank" 새 창/새 탭에서 열기
  • target="_self" 현재 창에서 열기
  • download 파일 다운로드 지정

4. 멀티미디어 & 입력 태그

  • <img> 이미지 삽입 태그
  • src 이미지/파일 경로
  • alt 대체 텍스트 설명
  • <video> 비디오 재생 태그
  • <audio> 오디오 재생 태그
  • controls 재생 컨트롤러 표시
  • autoplay 자동 재생 설정
  • loop 반복 재생 설정
  • <input> 입력창 생성 태그
  • type 입력 양식 종류
  • type="text" 일반 텍스트 입력
  • type="password" 비밀번호 입력 (숨김)
  • type="checkbox" 다중 선택 체크박스
  • type="radio" 단일 선택 라디오
  • type="color" 색상 선택 도구
  • type="range" 슬라이더 조절 바
  • type="date" 날짜 선택 달력
  • type="file" 파일 업로드 선택
  • type="submit" 양식 전송 버튼
  • placeholder 입력 가이드 문구
  • required 필수 입력 항목 설정
  • readonly 읽기 전용 설정
  • disabled 사용 불가(비활성) 설정
  • <button> 클릭 버튼 태그
  • <select> 드롭다운 선택창
  • <option> 드롭다운 항목

5. 표(Table) 태그

  • <table> 표 전체 구역
  • <tr> 표의 한 줄 (행)
  • <th> 표의 제목 칸
  • <td> 표의 일반 칸
  • <thead> 표 상단(제목) 구역
  • <tbody> 표 본문 구역
  • border="1" 표 테두리 두께 설정

6. 구조 & 섹션 태그 (Semantic)

  • <div> 블록 구역 나누기
  • <header> 상단 영역 (로고/메뉴)
  • <nav> 내비게이션 메뉴
  • <main> 메인 콘텐츠 영역
  • <section> 주제별 섹션 구역
  • <article> 독립적 콘텐츠 구역
  • <aside> 사이드바/부가 정보
  • <footer> 하단 영역 (정보/카피)

7. 공통 속성 (Global Attributes)

  • class 그룹 스타일 이름
  • id 고유 식별 이름
  • title 마우스 오버 툴팁
  • style 인라인 CSS 디자인
  • hidden 요소 숨김 처리
  • data-* 사용자 정의 데이터 저장
