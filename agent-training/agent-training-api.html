허허, 여기까지 오다니 대단하군! ⚡
기초 훈련을 통해 네 컴퓨터 안에서의 마법은 완벽해졌어. 이제 인터넷 너머 머나먼 외부 세상(서버)과 통신하는 진짜 심화 마법을 가르쳐주마!
우리의 목표는 강아지 나라 본부에 요원(fetch)을 파견해 귀여운 강아지 사진을 받아오는 거란다.
기초에서 배웠던 '마법 버튼'을 누를 때마다 새로운 사진을 받아오도록 코드를 합쳐볼 거야.
준비됐다면 우측 편집기에서 심화 미션을 시작해 보렴!
📘 정보국 1급 기밀 (JS 마법 백과사전)
심화 미션 1

마법 버튼에 시동 걸기

인터넷 너머로 통신을 보낼 때는 언제 보낼지 타이밍이 중요해. 우리는 버튼을 누를 때마다 사진을 가져올 거야!

[1] btn.onclick = function() { } 뼈대를 만들고
[2] 괄호 안에 alert("강아지 본부 연결 시도!");를 넣어 버튼이 살아있는지 콕 찔러보렴!

힌트: 우측 편집기에 뼈대는 미리 준비해 두었단다!
alert("강아지 본부 연결 시도!");
심화 미션 2

인터넷으로 통신 요원 파견하기 (fetch)

이제 진짜 인터넷 세상인 강아지 나라 본부로 통신 요원(fetch)을 파견해 보자!

[1] 미션 1에서 썼던 alert 코드는 지우고
[2] 그 자리에 fetch(...) 명령을 적어 요원을 파견해 보렴!

힌트: 괄호 안에 주소를 정확히 넣어야 요원이 길을 잃지 않아!
fetch('https://dog.ceo/api/breeds/image/random')

💡 어? 아무 일도 안 일어나는데요?

요원이 무사히 통신을 하고 돌아왔지만, 화면을 바꾸라고 명령하지 않았기 때문이야. 키보드에서 F12(개발자 도구)를 누르고 Network(네트워크) 탭을 연 뒤 버튼을 다시 눌러봐! 진짜로 통신이 슝~ 날아가는 게 보일 거야.

심화 미션 3

돌아온 배낭 해독하기 (.then 과 json)

요원이 성공적으로 돌아왔을 때(.then) 가져온 응답 배낭(response)은 꽁꽁 암호화되어 있어. 자바스크립트가 읽을 수 있게 번역(json)부터 하자!

[1] fetch 코드 바로 밑줄에 .then(...) 을 이어 붙이기

힌트: 마침표(.)로 꼬리에 꼬리를 무는 방식(체이닝)이야!
.then(function(response) { return response.json(); })
심화 미션 4

엑스레이 안경으로 진짜 데이터 관찰하기! (console)

해독이 끝난 진짜 상자(data) 안에는 무엇이 들어있을까? 섣불리 사진을 바꾸기 전에, 엑스레이 안경(console.log)을 쓰고 내용물을 직접 확인해 보자!

[1] 미션 3 코드 밑에 또다시 .then(...) 을 이어 붙이고
[2] 중괄호 안에 console.log(data); 를 넣어 콘솔창에 찍어보기!

힌트: 코드를 적고 버튼을 누른 뒤, F12 Console(콘솔) 탭을 확인해봐!
.then(function(data) { console.log(data); // 배낭 속을 직접 훔쳐보자! alert("새 강아지 사진을 찾았습니다! 🐶"); // 👇 미션 5: 이 아래에 사진을 갈아 끼우는 마법을 넣을 거야! })

✨ 데이터 구조 분석 (JSON)

콘솔창에 {message: '강아지주소', status: 'success'} 라고 찍힌 게 보이니? 이게 바로 서버가 우리에게 준 진짜 데이터(JSON)의 생김새란다! 이름표(message)와 내용물(주소)이 짝을 이루고 있지?

심화 미션 5

분석한 데이터를 바탕으로 사진 액자 갈아 끼우기!

이제 데이터 안에 message라는 이름표로 강아지 주소가 들어있다는 걸 눈으로 확인했지? 확신을 가지고 화면의 사진을 갈아 끼워 보자!

[1] 미션 4 코드 안의 '미션 5' 주석 아랫줄 빈 공간에
[2] 화면 사진(main-img)의 주소(src)를 data.message로 갈아 끼우는 마법을 추가해!

힌트: 미션 4의 중괄호가 닫히기 전에 그 안에 추가해야 해!
document.getElementById('main-img').src = data.message;
심화 미션 6 (최종 완성)

실패에 대비하는 든든한 방패 (.catch)

인터넷 세상에서는 언제든 서버가 고장 나거나 랜선이 뽑힐 수 있어. 에러가 나면 훈련생들이 당황하지 않게 방패(.catch)를 세워 알림을 띄워주자!

[1] 맨 아래 꼬리에 .catch(...) 를 이어 붙이기
[2] 마법 버튼을 눌러서 강아지 사진이 계속 바뀌는지 최종 완성 확인!

힌트: 맨 마지막 꼬리에 catch를 달아주면 완벽해!
.catch(function(error) { alert("통신 실패! 😭 인터넷을 확인하세요."); });
