HTML과 CSS 요원들이 예쁜 뼈대와 디자인을 만들어 주었구만! 하지만 아직 움직이지 않는 그림일 뿐이지.
이제 내 차례야! 버튼을 누르면 알림이 뜨고, 텍스트가 변하는 등 움직이고 반응하는 진짜 웹페이지를 만들어 볼 거란다.
어렵지 않으니까 아저씨의 마법 주문을 잘 따라 해보렴! 우측 편집기에서 미션 시작!
📘 정보국 1급 기밀 (JS 마법 백과사전)
심화 미션 1

접속 성공 주문서 작성하기

자바스크립트 마법사는 내 컴퓨터 밖의 머나먼 서버와도 대화할 수 있단다! 우선 강아지 나라 본부와 통신이 잘 연결되었을 때 실행할 주문서를 써보자.

[1] showSuccess 라는 이름의 주문서를 만들고 괄호 안에 response 라고 적어두기
[2] 주문서 안에 alert를 써서 상태 번호(response.status)가 포함된 성공 메시지 띄우기

힌트: function과 alert 사용 (배낭에서 상태 번호인 response.status를 꺼내 더해줘요!)
function showSuccess(response) { alert("강아지 나라 서버 접속 성공! 📡 (상태 코드: " + response.status + ")"); }

✨ JS 작동 원리 탐구

· 매개변수 비유 (response): 괄호 안의 response는 요원이 서버에서 가져올 '응답 배낭'이란다! 미리 적어두는 건 배낭을 놓을 '선반'을 마련해 두는 셈이지.
· 점(.) 찍고 꺼내기: 배낭 뒤에 점(.)을 찍으면 안의 내용물을 꺼낼 수 있어. response.status는 통신 성공 여부를 알려주는 상태 번호(보통 200)란다.
· 마법의 풀칠 (+ 연산자): 텍스트와 숫자 사이에 +를 쓰면 수학의 덧셈이 아니라 두 내용을 하나로 딱 붙여준단다!

심화 미션 2

인터넷 너머로 통신 요원 파견하기

이제 강아지 나라 서버 주소로 요원(fetch)을 파견하고, 안전하게 돌아오면 방금 만든 주문서를 발동시켜 보자!

[1] fetch 명령으로 서버 주소에 요원을 파견하고
[2] 요원이 돌아오면(.then) 방금 만든 showSuccess 주문서를 발동하도록 연결하기

힌트: fetch와 .then 사용 (괄호 없이 함수 이름만 쓱 연결해요!)
fetch('https://dog.ceo/api/breeds/image/random') .then(showSuccess);

💡 자동 배달 원리

.then(showSuccess)라고 괄호 없이 적어두면, 요원이 돌아와서 자기가 들고 온 응답 배낭(response)을 심화 미션 1에서 만들어둔 선반 위에 알아서 챡! 올려놓게 된단다!

심화 미션 3

돌아온 데이터 가방 해독하기 (번역기 주문서)

요원이 들고 온 응답 배낭(response)은 꽁꽁 잠겨 있어 바로 읽을 수 없단다. 자바스크립트가 읽을 수 있도록 번역 주문서를 써보자!

[1] translateData 라는 이름의 번역기 주문서를 만들고
[2] response.json() 명령으로 내용물을 번역한 뒤 반환(return)하기

힌트: function과 return, response.json() 사용
function translateData(response) { return response.json(); }

💡 토스(Toss) 마법

번역 완료 후 적는 return은 번역이 끝난 진짜 내용물인 '강아지 선물 상자'를 다음 대기 중인 주문서에게 휙! 던져(Toss)주는 마법 손길이란다.

심화 미션 4 (최종 완성)

해독한 강아지 사진을 액자에 걸기!

마지막으로, 번역기가 휙 던져준 진짜 강아지 상자(data)를 받아 화면의 사진 액자를 갈아 끼워주는 주문서를 완성해 보자!

[1] hangPicture 라는 이름의 주문서를 만들고 상자(data) 받기
[2] 상태가 "success"인지 깐깐한 저울(===)로 확인하고
[3] 맞다면 알림창을 띄우고 메인 사진(main-img)의 주소(src)를 data.message로 갈아 끼우기
[4] 맨 밑에 fetch부터 .then까지 3단 콤보 마법 발동시키기!

힌트: function, if, src 속성을 활용해 3단 콤보 마법을 최종 완성해요!
function hangPicture(data) { if (data.status === "success") { alert("실시간 강아지 사진 액자 장착! 🐶"); document.getElementById('main-img').src = data.message; } } // 최종 3단 콤보 마법 발동! (파견 -> 번역 -> 사진 걸기) fetch('https://dog.ceo/api/breeds/image/random') .then(translateData) .then(hangPicture);

✨ JS 작동 원리 탐구

· 선물 상자 받기: 괄호 속에 data라고 써두는 것은, 앞 단계 번역기 주문서가 던져준 선물 상자를 안전하게 받기 위해 '두 손'을 앞으로 내밀어 두는 것이란다!
· 깐깐한 저울 (===): 양쪽이 똑같은지 확인하는 저울이야. 모양뿐만 아니라 타입(텍스트인지 숫자인지)까지 완벽히 같아야만 참(True)으로 인정해주지.
