기초 훈련을 통해 네 컴퓨터 안에서의 마법은 완벽해졌어. 이제 인터넷 너머 머나먼 외부 세상(서버)과 통신하는 진짜 심화 마법을 가르쳐주마!
우리의 목표는 강아지 나라 본부에 요원(fetch)을 파견해 귀여운 강아지 사진을 받아오는 거란다.
기초에서 배웠던 '마법 버튼'을 누를 때마다 새로운 사진을 받아오도록 코드를 합쳐볼 거야.
준비됐다면 우측 편집기에서 심화 미션을 시작해 보렴!

심화 미션 1

마법 버튼에 시동 걸기

인터넷 너머로 통신을 보낼 때는 언제 보낼지 타이밍이 중요해. 우리는 버튼을 누를 때마다 사진을 가져올 거야!

[1] dogBtn.onclick = function() { } 뼈대를 만들고
[2] 괄호 안에 alert("강아지 본부 연결 시도!");를 넣어 버튼이 살아있는지 콕 찔러보렴!

힌트: alert 사용 (알림창 띄우기)
alert("강아지 본부 연결 시도!");

✨ 팁: 우측 편집기에 뼈대는 미리 준비해 두었단다!

심화 미션 2

인터넷으로 통신 요원 파견하기 (fetch)

이제 진짜 인터넷 세상인 강아지 나라 본부로 통신 요원(fetch)을 파견해 보자!

[1] 미션 1에서 썼던 alert 코드는 지우고
[2] 그 자리에 fetch(...) 명령을 적어 요원을 파견해 보렴!

힌트: fetch 사용 (외부 서버 통신)
fetch('https://dog.ceo/api/breeds/image/random')

✨ 팁: 괄호 안에 주소를 정확히 넣어야 요원이 길을 잃지 않아!

💡 어? 아무 일도 안 일어나는데요?

요원이 무사히 통신을 하고 돌아왔지만, 화면을 바꾸라고 명령하지 않았기 때문이야. 키보드에서 F12(개발자 도구)를 누르고 Network(네트워크) 탭을 연 뒤 버튼을 다시 눌러봐! 진짜로 통신이 슝~ 날아가는 게 보일 거야.

✨ JS 작동 원리 탐구: 숨겨진 GET 명령표

우리가 방금 쓴 fetch(주소)는 사실 뒤에 무언가가 생략되어 있단다. 원래는 fetch(주소, { method: 'GET' }) 이라고 적어야 해! 아무 말 없이 심부름을 보내면, 요원은 기본적으로 데이터를 '가져오기(GET)'만 한다는 사실을 기억해 둬!

심화 미션 3

돌아온 배낭 해독하기 (.then 과 json)

요원이 성공적으로 돌아왔을 때(.then) 가져온 응답 배낭(response)은 꽁꽁 암호화되어 있어. 자바스크립트가 읽을 수 있게 번역(json)부터 하자!

[1] fetch 코드 바로 밑줄에 .then(...) 을 이어 붙이기

힌트: .then, .json 사용 (배낭 해독하기)
.then(function(response) { return response.json(); })

✨ 팁: 마침표(.)로 꼬리에 꼬리를 무는 방식(체이닝)이야!

✨ JS 작동 원리 탐구: 기다림의 마법 (.then)

인터넷 너머 서버로 보낸 요원이 언제 돌아올지 우리는 정확히 알 수 없어. 1초가 걸릴 수도, 5초가 걸릴 수도 있지! 그래서 .then(...)이라는 '기다림의 마법'을 쓰는 거야. "요원이 돌아올 때까지 기다렸다가, 도착하면 이 중괄호 { } 안의 명령을 실행해 줘!" 라는 뜻이란다.

✨ JS 작동 원리 탐구: 괄호 안의 response는 어떻게 들어온 걸까?

통신 요원(fetch)이 강아지 본부에서 돌아올 때 빈손으로 오지 않겠죠? 무언가 응답 배낭을 메고 옵니다. .then 마법이 발동될 때, 요원이 메고 온 그 배낭을 function(response)의 괄호 안으로 쏙! 하고 던져주는(넘겨주는) 것이랍니다. 그래서 우리는 response라는 이름으로 그 배낭(결과값)을 받아 열어볼 수 있는 거예요!

심화 미션 4

엑스레이 안경으로 진짜 데이터 관찰하기! (console)

해독이 끝난 진짜 상자(data) 안에는 무엇이 들어있을까? 섣불리 사진을 바꾸기 전에, 엑스레이 안경(console.log)을 쓰고 내용물을 직접 확인해 보자!

[1] 미션 3 코드 밑에 또다시 .then(...) 을 이어 붙이고
[2] 중괄호 안에 console.log(data); 를 넣어 콘솔창에 찍어보기!

힌트: .then, console.log 사용 (데이터 관찰)
.then(function(data) { console.log(data); // 배낭 속을 직접 훔쳐보자! alert("새 강아지 사진을 찾았습니다! 🐶"); // 👇 미션 5: 이 아래에 사진을 갈아 끼우는 마법을 넣을 거야! })

✨ 팁: 코드를 적고 버튼을 누른 뒤, F12 Console(콘솔) 탭을 확인해봐!

✨ 데이터 구조 분석 (JSON)

콘솔창에 {message: '강아지주소', status: 'success'} 라고 찍힌 게 보이니? 이게 바로 서버가 우리에게 준 진짜 데이터(JSON)의 생김새란다! 이름표(message)와 내용물(주소)이 짝을 이루고 있지?

심화 미션 5

분석한 데이터를 바탕으로 사진 액자 갈아 끼우기!

이제 데이터 안에 message라는 이름표로 강아지 주소가 들어있다는 걸 눈으로 확인했지? 확신을 가지고 화면의 사진을 갈아 끼워 보자!

[1] 미션 4 코드 안의 '미션 5' 주석 아랫줄 빈 공간에
[2] 화면 사진(main-img)의 주소(src)를 data.message로 갈아 끼우는 마법을 추가해!

힌트: document.getElementById, src 사용 (이미지 주소 교체)
document.getElementById('main-img').src = data.message;

✨ 팁: 미션 4의 중괄호가 닫히기 전에 그 안에 추가해야 해!

심화 미션 6

실패에 대비하는 든든한 방패 (.catch)

인터넷 세상에서는 언제든 서버가 고장 나거나 랜선이 뽑힐 수 있어. 에러가 나면 훈련생들이 당황하지 않게 방패(.catch)를 세워 알림을 띄워주자!

[1] 맨 아래 꼬리에 .catch(...) 를 이어 붙이기
[2] 마법 버튼을 눌러서 강아지 사진이 계속 바뀌는지 최종 완성 확인!

힌트: .catch 사용 (에러 방어막)
.catch(function(error) { alert("통신 실패! 😭 인터넷을 확인하세요."); });

✨ 팁: 맨 마지막 꼬리에 catch를 달아주면 완벽해!

심화 미션 7

마법 주문 압축하기 (화살표 함수)

자바스크립트 마법사들은 코드를 짧고 깔끔하게 쓰는 걸 좋아해! 최신 마법인 화살표 함수(Arrow Function)를 써서 코드를 압축해 보자.

[1] 미션 3 코드에서 function 글자를 지우고, 괄호 뒤에 => 화살표를 넣어줘!
[2] 괄호 안의 내용이 한 줄뿐일 때는 중괄호 { } 와 return 글자도 지워서 완벽한 한 줄로 줄일 수 있단다!

🚨 주의: 아래 코드를 복사해서 붙여넣지 말고, 에디터에 적어둔 기존 코드에서 글자들을 직접 지우고 수정하면서 모양을 바꿔보렴!

힌트: 기존 코드에서 function 지우기, => 넣기, 중괄호와 return 지우기
// 1단계 압축: 기존 코드의 function을 지우고 화살표 넣기! .then((response) => { return response.json(); }) // 2단계 압축: 중괄호와 return도 생략! (완벽한 한 줄 마법 ✨) .then(response => response.json())

💡 추가 미션 1: 미션 4, 5가 들어있는 두 번째 .then(...) 부분도 화살표 함수로 압축해 볼까? (이곳은 명령이 여러 줄이므로 1단계까지만 압축할 수 있어!)

💡 추가 미션 2: 미션 6의 .catch(...) 방어막 부분도 화살표 함수로 압축해 볼까? (이곳은 코드가 한 줄뿐이므로 완벽하게 2단계까지 압축할 수 있어!)

심화 미션 8

고양이 요원 파견하고 엑스레이(console)로 관찰하기!

이제 고양이 본부(The Cat API)로 통신 요원을 파견할 거야! 강아지 본부와 다르게 고양이 본부의 응답 배낭은 어떻게 생겼는지 콘솔창(F12)을 열어 먼저 확인해보자.

[1] 편집기 아래쪽 '미션 8' 구역에 요원 파견 및 해독 코드를 적어봐!
[2] console.log(data); 를 적고 고양이 버튼을 눌러 콘솔창을 살펴보기

힌트: fetch, .then, console.log 사용 (고양이 통신 파견)
alert("고양이 본부 연결 시도! 🐱"); fetch('https://api.thecatapi.com/v1/images/search') .then(response => response.json()) .then(data => { console.log(data); // 앗! 대괄호[ ]로 싸여있네? alert("새 고양이 사진을 찾았습니다! 🐱"); // 👇 미션 9: 이 아래에 0번째 상자를 열어서 사진을 꺼내는 마법을 넣을 거야! })

✨ 팁: 뼈대는 강아지 때랑 똑같아! 주소만 고양이 본부야.

💡 추가 미션: 통신 실패 방어막(.catch) 달기

강아지 때처럼 맨 마지막 꼬리에 통신 실패를 막아줄 방어막을 잊지 말고 달아보렴!

힌트: .catch 사용 (에러 방어막 추가)
.catch(error => alert("고양이 통신 실패! 😭 인터넷을 확인하세요."));

✨ 팁: 미션 8 코드 맨 밑줄에 아래 방어막 코드를 그대로 복사해서 이어 붙이면 돼!

심화 미션 9 (최종 완성)

배열 상자 열어서 고양이 사진 꺼내기!

대괄호 [ ] 상자 안에 든 내용물을 꺼내려면, 컴퓨터에게 몇 번째 상자를 열어야 하는지 알려줘야 해. 컴퓨터는 숫자를 0부터 센다는 사실을 기억해!

[1] 미션 8 코드 안의 '미션 9' 주석 아랫줄 빈 공간에
[2] 화면 사진(main-img)을 data[0].url로 갈아 끼우는 마법을 추가해!

힌트: 배열 인덱스[0], src 사용 (0번째 상자 꺼내기)
document.getElementById('main-img').src = data[0].url;

✨ 팁: 미션 8의 중괄호가 닫히기 전에 그 안에 추가해야 해!

✨ 배열(Array) 데이터 파악하기

콘솔창을 보니 이번엔 [ { url: '고양이주소' } ] 처럼 바깥쪽이 대괄호 [ ] 로 한 번 더 싸여있는 걸 발견했니? 이걸 '배열(Array)' 상자라고 부른단다. 여러 개의 데이터가 기차처럼 줄지어 들어올 때 쓰는 특별한 상자야!

심화 미션 10

비밀 방명록 남기기 (POST)

사진을 잘 봤다면 이제 방명록을 남겨볼까? 요원에게 데이터를 단순히 가져오는(GET) 게 아니라, 새로 만들어 달라고(POST) 요청할 수 있단다!

[1] 편집기 맨 아래쪽 '미션 10' 구역을 찾아보렴!
[2] { method: 'POST' } 라는 특별 임무와 함께, 우리가 쓴 글(body)을 쥐여 보내자.

힌트: method: 'POST', body: JSON.stringify(...) 사용
const message = document.getElementById("guestbook-input").value; fetch('/api/guestbook', { method: 'POST', body: JSON.stringify({ text: message }) }) .then(response => response.json()) .then(data => alert("방명록이 등록되었습니다! (ID: " + data.id + ")"));

✨ 팁: 진짜 비밀 서버라서 새로고침 버튼을 누르면 서버에 저장된 내 글을 가져올 거야!

심화 미션 11

방명록 불러오기 (GET)

우리가 방금 등록한 방명록이 서버에 잘 저장되었는지 가져와서(GET) 화면에 그려보자!

[1] '미션 11' 구역에 fetch(GET) 마법을 적어봐.
[2] 가져온 데이터를 리스트에 차례대로 그려주자!

힌트: fetch('/api/guestbook') 사용
fetch('/api/guestbook') .then(response => response.json()) .then(data => { const list = document.getElementById("guestbook-list"); list.innerHTML = ""; // 기존 목록 비우기 data.forEach(post => { list.innerHTML += "<li>" + post.title + "</li>"; }); });

✨ 팁: 서버에서 가져온 글들을 반복문(forEach)을 이용해 하나씩 꺼내 화면에 그려주는 거란다!

심화 미션 12

내가 쓴 흔적 지우기 (DELETE)

방명록이 마음에 들지 않아? 그럼 요원에게 화면의 방명록 리스트를 지워 달라고(DELETE) 요청해 보자!

[1] 편집기 맨 아래쪽 '미션 12' 구역을 찾아보렴!
[2] { method: 'DELETE' } 라는 특별 임무를 주고 화면도 초기화하자!

힌트: method: 'DELETE', innerHTML = "" 사용
fetch('/api/guestbook', { method: 'DELETE' }) .then(response => { alert("서버에서 흔적이 삭제되었습니다! 화면도 지울게! 🗑️"); document.getElementById("guestbook-list").innerHTML = ""; });

✨ 팁: 통신이 끝난 후 화면의 리스트 안쪽(innerHTML)을 아무것도 없는 텅 빈 상태("")로 만들면 화면도 감쪽같이 지워진단다!
