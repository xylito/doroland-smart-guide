허허, 반가워 신입 요원! ⚡
나는 정적인 웹페이지에 생명을 불어넣는 마법사, JS(자바스크립트) 요원 아저씨란다.
HTML과 CSS 요원들이 예쁜 뼈대와 디자인을 만들어 주었구만! 하지만 아직 움직이지 않는 그림일 뿐이지.
이제 내 차례야! 버튼을 누르면 알림이 뜨고, 글자가 변하는 등 움직이고 반응하는 진짜 웹페이지를 만들어 볼 거란다.
어렵지 않으니까 아저씨의 마법 주문을 잘 따라 해보렴! 우측 편집기에서 미션 시작!
📘 정보국 1급 기밀 (JS 마법 백과사전)
미션 1

마법 지팡이로 제목 찾기

HTML에 있는 큰 제목(main-title)을 찾아서 'title'이라는 상자에 담아보자꾸나!

💡 상수 const: const는 절대 열리지 않는 '안전한 보관함'이란다! 한 번 이름을 붙여서 무언가를 담아두면, 다른 마법사가 와도 내용물을 함부로 바꿀 수 없지.

힌트: document.getElementById 사용
const title = document.getElementById('main-title');

💡 추가 미션: 다른 지팡이(추적기) 종류도 구경해 보렴!

· getElementById('ID') : 특정 ID로 찾기

· getElementsByTagName('태그') : 태그 이름으로 모두 찾기

· getElementsByClassName('클래스') : 클래스 이름으로 모두 찾기

· querySelector('CSS선택자') : 만능 추적기 (첫 번째 요소만)

· querySelectorAll('CSS선택자') : 만능 복수 추적기 (모두 찾기)

- 선택자 예시: #main-title (ID), .content-box (클래스), h1 (태그)

미션 2

글자 바꾸기 마법

찾아낸 제목의 글자를 "자바스크립트 마법 성공!"으로 확 바꿔볼까?

힌트: innerText 사용
title.innerText = "자바스크립트 마법 성공!";

💡 추가 미션: 다른 방식으로 글자나 태그를 제어해 보렴!

힌트: innerHTML 사용
title.innerHTML = "<i>자바스크립트 마법 성공!</i>";
미션 3

기본 모양 및 색상 마법

제목이 눈에 확 띄도록 주황색(orange)으로 색을 입혀주렴!

힌트: style.color 사용
title.style.color = "orange";

💡 추가 미션: 다른 모양과 색상을 제어하는 주문들도 한 줄씩 외쳐보렴!

힌트 1: style.backgroundColor 사용 (배경색 칠하기)
title.style.backgroundColor = "pink"; 힌트 2: style.border 사용 (테두리 두르기)
title.style.border = "2px solid orange"; 힌트 3: style.borderRadius 사용 (모서리 둥글게 깎기)
title.style.borderRadius = "15px"; 힌트 4: style.padding 사용 (안쪽 여백 늘리기)
title.style.padding = "20px";
미션 4

글자 크기 및 정렬 마법

제목의 크기를 25픽셀(25px)로 바꾸어보자꾸나!

힌트: style.fontSize 사용
title.style.fontSize = "25px";

💡 추가 미션: 텍스트 레이아웃을 조절하는 주문들도 한 줄씩 실행해 보렴!

힌트 1: style.textAlign 사용 (글자 가운데 정렬)
title.style.textAlign = "left"; 힌트 2: style.textShadow 사용 (예쁜 그림자 넣기)
title.style.textShadow = "2px 2px 4px rgba(0,0,0,0.2)"; 힌트 3: style.boxShadow 사용 (예쁜 그림자 넣기)
title.style.boxShadow = "2px 2px 4px rgba(0,0,0,0.2)";
미션 5

신입 요원 종합 시험 (4단 콤보 마법)

허허, 한 단계씩 따라 하는 건 너무 쉬웠지? 이제 실력을 증명할 때다!
문장 전체가 아니라 <span> 요소인 한 단어만 작업해볼거다!

[1] 아이디가 park-status인 요소를 찾아
[2] 글자를 '정상 운영'으로 바꾸고
[3] 파란색(blue) 색상을 입힌 후
[4] 글자 크기를 20픽셀(20px)로 시원하게 키워라!

4단 콤보 종합 시험 마법을 완성해 보렴!

힌트: 미션 1, 2, 3, 4 단계의 기술을 멋지게 하나로 엮어봐요!
const statusSpan = document.getElementById('park-status'); statusSpan.innerText = "정상 운영"; statusSpan.style.color = "blue"; statusSpan.style.fontSize = "20px";
미션 6

마우스 커서 모양 바꾸기

마치 진짜 링크 단추인 것처럼 제목 위에 마우스가 올라갔을 때 손가락 모양(pointer)으로 포인터를 바꿔보렴!

힌트: style.cursor 사용
title.style.cursor = "pointer";

💡 추가 미션: 다른 마우스 커서 모양들도 구경해 보렴!

· title.style.cursor = "help"; : 도움말 모양 (물음표)

· title.style.cursor = "wait"; : 로딩 중 (모래시계/빙글빙글)

· title.style.cursor = "crosshair"; : 십자선 (조준점)

· title.style.cursor = "not-allowed"; : 금지 모양

미션 7

마우스오버 생동감 종합 마법 (글자 + 크기 + 회전)

먼저 스타일 변화가 부드럽게 일어나도록 트랜지션(style.transition = "all 0.5s ease";)을 걸어주자!
그 다음, 마우스가 올라갔을 때(onmouseover) 글자 내용을 "어라? 나를 만졌네! 😮"로 바꿈과 동시에 1.1배 크기 및 시계방향 5도 회전(style.transform = "scale(1.1) rotate(5deg)";)을 시키고, 마우스가 떠났을 때(onmouseout) 글자 내용을 "👋 JavaScript 요원 다녀감 😆"로 바꾸며 원래 상태로 되돌리는 초특급 생동감 마법을 완성해 보렴!

힌트: style.transition 설정 후, onmouseover와 onmouseout, style.transform 사용
title.style.transition = "all 0.5s ease"; title.onmouseover = function() { title.innerText = "어라? 나를 만졌네! 😮"; title.style.transform = "scale(1.1) rotate(5deg)"; }; title.onmouseout = function() { title.innerText = "👋 JavaScript 요원 다녀감 😆"; title.style.transform = ""; };

💡 JS 요원 아저씨의 마법 돋보기 (작동 원리):

· 부드러운 전환 (transition): 미리 transition 효과를 걸어두었기 때문에, 마우스를 갖다 대거나 뗄 때 `transform` 마법이 뚝뚝 끊기지 않고 영화처럼 스르륵 부드럽게 연출된단다!

· 원상 복구 ("" 빈 문자열): scale(1.1)은 1.1배 크기 확대, rotate(5deg)는 5도 회전이란다. 마우스 아웃 상태일 때는 제자리로 되돌리기 위해 빈 문자열("")을 대입하면, 추가했던 인라인 마법이 싹 지워지면서 원래 CSS 디자인 상태로 깔끔하게 돌아간단다!

미션 8

사진 크기 조절하기 (마우스 이벤트 복습)

이번에는 사진(아이디: main-img)을 찾아서, 마우스가 올라갔을 때(onmouseover) 1.2배로 커지고(scale(1.2)), 마우스가 떠났을 때(onmouseout) 원래 크기로 돌아오는 마법을 복습해 보자꾸나!

힌트: document.getElementById, onmouseover, onmouseout 사용
const img = document.getElementById('main-img'); img.onmouseover = function() { img.style.transform = "scale(1.2)"; }; img.onmouseout = function() { img.style.transform = ""; };
미션 9

알림 마법 주문서(함수) 작성하기

화면에 환영 알림창을 띄우는 showMagic 이라는 주문서를 미리 써보자꾸나!
⚠️ 중요: 마법 주문서(함수)를 적어두는 것만으로는 마법이 바로 나가진 않는단다. 그래서 지금은 코드를 완성해도 결과 화면에 아무 일도 일어나지 않는 게 정상이란다! 다음 미션 10에서 이 주문서를 발동할 거란다.

힌트: function과 alert 사용
function showMagic() { alert("환영합니다!"); }
미션 10

버튼을 누를 때 마법 주문 발동시키기

마법 버튼(아이디: magic-btn)을 찾아내서, 클릭할 때 방금 만든 showMagic 주문서가 샥 발동되도록 연결해 보자!
⚠️ 특급 주의: 주문 이름 뒤에 괄호()를 붙이면 그 자리에서 즉시 주문이 깨져버려요! 버튼을 '누를 때'만 발동하게 하려면 괄호 없이 주문 이름인 showMagic만 깔끔하게 연결해야 한단다.

힌트: document.getElementById 와 onclick 사용
const btn = document.getElementById('magic-btn'); btn.onclick = showMagic;
미션 11 (보너스)

접속 성공 주문서 작성하기

자바스크립트 마법사는 내 컴퓨터 밖의 머나먼 서버와도 대화할 수 있단다!
우선 강아지 나라 본부와 통신이 잘 연결되었을 때 실행할 showSuccess 주문서를 써보자.
💡 매개변수 비유: 괄호 안의 response는 요원이 등 뒤에 메고 올 '응답 배낭(상자)'이란다. 주문을 적을 때 괄호에 response라고 써두는 건, 나중에 요원이 가져올 배낭을 내려놓을 '배낭 선반'을 미리 마련해 두는 것이란다!
🔑 점('.') 찍고 꺼내기 실제 활용: 배낭 뒤에 **점(.)**을 찍어서 통신이 잘 성공했는지 알려주는 서버 상태 번호인 response.status를 쏙 꺼내서 화면에 띄워보자! 연결이 성공하면 마법의 숫자인 200을 돌려받게 된단다!
🧪 문법 탐험 (+ 연산자): 글자와 글자(또는 숫자) 사이에 +를 쓰면 수학의 덧셈이 아니라, 두 글자를 딱 붙여주는 '마법의 풀칠' 역할을 한단다!

힌트: function과 alert 사용 (배낭에서 상태 번호인 response.status를 꺼내 더해줘요!)
function showSuccess(response) { alert("강아지 나라 서버 접속 성공! 📡 (상태 코드: " + response.status + ")"); }
미션 12

인터넷 너머로 통신 요원 파견하기

이제 강아지 나라 서버 주소로 요원(fetch)을 파견하고, 요원이 안전하게 돌아오면(.then) 방금 만든 showSuccess 주문서를 발동하도록 이름만 쓱 전해주자!
💡 자동 배달 원리: .then(showSuccess)라고 괄호 없이 적어두면, 요원이 돌아와서 자기가 들고 온 응답 배낭(response)을 우리가 미션 13에서 만들어둔 배낭 선반 괄호 위에 알아서 챡! 올려놓게 된단다!

힌트: fetch와 .then 사용 (괄호 없이 함수 이름만 쓱 연결해요!)
fetch('https://dog.ceo/api/breeds/image/random') .then(showSuccess);
미션 13

돌아온 데이터 가방 해독하기 (번역기 주문서)

요원이 들고 온 응답 배낭(response)은 꽁꽁 잠겨 있어 바로 읽을 수 없단다. 배낭 속 내용물을 자바스크립트가 읽을 수 있도록 번역해 주는 translateData 주문서를 새로 작성해 보자!
💡 토스(Toss) 비유: 번역 완료 후 적는 return response.json();의 return은 번역이 끝난 진짜 내용물인 '강아지 선물 상자'를 다음 대기 중인 주문서에게 휙! 던져(Toss)주는 마법 손길이란다.

힌트: function과 return, response.json() 사용
function translateData(response) { return response.json(); }
미션 14 (최종 완성)

해독한 강아지 사진을 액자에 걸기!

마지막 단계로, 앞서 번역기가 휙 던져준 진짜 강아지 선물 상자(data)를 받아 화면의 사진 액자(아이디: main-img)의 사진 주소(src)로 갈아 끼워주는 주문서(hangPicture)를 완성해 보렴!
💡 선물 상자 받기: 주문서 괄호 속에 data라고 써두는 것은, 앞 단계 번역기 주문서가 던져준 선물 상자(data)를 안전하게 받기 위해 '두 손'을 앞으로 쭉 내밀어 두는 것이란다!
⚖️ 문법 탐험 (===): === 기호는 양쪽이 똑같은지 확인하는 '깐깐한 저울'이란다! 모양뿐만 아니라 타입(글자인지 숫자인지)까지 똑같아야만 참(True)이라고 인정해주지.

힌트: function, if, src 속성을 활용해 3단 콤보 마법을 최종 완성해요!
function hangPicture(data) { if (data.status === "success") { alert("실시간 강아지 사진 액자 장착! 🐶"); document.getElementById('main-img').src = data.message; } } // 최종 3단 콤보 마법 발동! (파견 -> 번역 -> 사진 걸기) fetch('https://dog.ceo/api/breeds/image/random') .then(translateData) .then(hangPicture);
