📜 임무 지령서
허허, 반가워 신입 요원! ⚡
나는 정적인 웹페이지에 생명을 불어넣는 마법사, JS(자바스크립트) 요원 아저씨란다.
HTML과 CSS 요원들이 예쁜 뼈대와 디자인을 만들어 주었구만! 하지만 아직 움직이지 않는 그림일 뿐이지.
이제 내 차례야! 버튼을 누르면 알림이 뜨고, 텍스트가 변하는 등 움직이고 반응하는 진짜 웹페이지를 만들어 볼 거란다.
어렵지 않으니까 아저씨의 마법 주문을 잘 따라 해보렴! 우측 편집기에서 미션 시작!
📘 정보국 1급 기밀 (JS 마법 백과사전)
미션 1

마법 지팡이로 제목 찾기

HTML에 있는 큰 제목(main-title)을 찾아서 'title'이라는 상자에 담아보자꾸나!

💡 상수 const: const는 절대 열리지 않는 '안전한 보관함'이란다! 한 번 이름을 붙여서 무언가를 담아두면, 다른 마법사가 와도 내용물을 함부로 바꿀 수 없지.

힌트: document.getElementById 사용
const title = document.getElementById('main-title');

💡 추가 미션: 다른 지팡이(추적기) 종류도 구경해 보렴!

· getElementById('ID') : 특정 ID로 찾기
· getElementsByTagName('태그') : 태그 이름으로 모두 찾기
· getElementsByClassName('클래스') : 클래스 이름으로 모두 찾기
· querySelector('CSS선택자') : 만능 추적기 (첫 번째 요소만)
· querySelectorAll('CSS선택자') : 만능 복수 추적기 (모두 찾기)
- 선택자 예시: #main-title (ID), .content-box (클래스), h1 (태그)

미션 2

텍스트 내용 마법

찾아낸 제목의 텍스트를 "자바스크립트 마법 성공!"으로 확 바꿔볼까?

힌트: innerText 사용 (텍스트 변경)
title.innerText = "자바스크립트 마법 성공!";

💡 추가 미션: 다른 방식으로 텍스트나 태그를 제어해 보렴!

힌트: innerHTML 사용 (텍스트 변경(태그))
title.innerHTML = "<i>자바스크립트 마법 성공!</i>";
미션 3

텍스트 스타일 마법

제목이 눈에 확 띄도록 주황색(orange)으로 색을 입혀주렴!

힌트: style.color 사용 (텍스트 색상 변경)
title.style.color = "orange";

💡 추가 미션: 다른 텍스트 꾸미기 주문들도 한 줄씩 외쳐보렴!

힌트 1: style.fontSize 사용 (텍스트 크기 변경)
title.style.fontSize = "25px"; 힌트 2: style.textAlign 사용 (텍스트 정렬 변경)
title.style.textAlign = "left"; 힌트 3: style.textShadow 사용 (텍스트 그림자 변경)
title.style.textShadow = "2px 2px 4px rgba(0,0,0,0.2)";
미션 4

배경 스타일 마법

이번엔 제목 배경 영역을 예쁜 핑크색(pink)으로 칠해보자꾸나!

힌트: style.backgroundColor 사용 (배경 색상 변경)
title.style.backgroundColor = "pink";

💡 추가 미션: 상자의 모양과 여백을 조절하는 주문들도 실행해 보렴!

힌트 1: style.padding 사용 (배경 내부 여백 변경)
title.style.padding = "20px"; 힌트 2: style.border 사용 (배경 테두리 변경)
title.style.border = "2px solid orange"; 힌트 3: style.borderRadius 사용 (배경 모서리 둥글게 변경)
title.style.borderRadius = "15px"; 힌트 4: style.boxShadow 사용 (배경 그림자 변경)
title.style.boxShadow = "2px 2px 4px rgba(0,0,0,0.2)";
미션 5

신입 요원 종합 시험 (4단 콤보 마법)

허허, 한 단계씩 따라 하는 건 너무 쉬웠지? 이제 실력을 증명할 때다!
문장 전체가 아니라 <span> 요소인 한 단어만 작업해볼거다!

[1] 아이디가 park-status인 요소를 찾아
[2] 텍스트를 '정상 운영'으로 바꾸고
[3] 텍스트 색상을 하얀색(white)으로 입힌 후
[4] 배경을 파란색(blue)으로 칠해라!

4단 콤보 종합 시험 마법을 완성해 보렴!

힌트: 미션 1, 2, 3, 4 단계의 기술을 하나로 엮어봐요!
const statusSpan = document.getElementById('park-status'); statusSpan.innerText = "정상 운영"; statusSpan.style.color = "white"; statusSpan.style.backgroundColor = "blue";
미션 6

커서 스타일 마법

마치 진짜 링크 단추인 것처럼 제목 위에 마우스가 올라갔을 때 손가락 모양(pointer)으로 포인터를 바꿔보렴!

힌트: style.cursor 사용
title.style.cursor = "pointer";

💡 추가 미션: "help"(물음표), "wait"(로딩중), "crosshair"(십자선), "not-allowed"(금지) 등 다양한 값을 넣어보렴!

미션 7

마우스오버 생동감 종합 마법 (텍스트 + 크기 + 회전)

이제 마우스 움직임에 맞춰 살아 움직이는 생동감 마법을 걸어보자!

[1] 마우스가 닿을 때(onmouseover) 텍스트와 크기, 회전을 바꾸고
[2] 마우스가 떠날 때(onmouseout) 텍스트를 바꾸며 원래 상태로 ("") 되돌려 보렴!

힌트: onmouseover, onmouseout 작성
title.onmouseover = function() { title.innerText = "어라? 나를 만졌네! 😮"; title.style.transform = "scale(1.1) rotate(5deg)"; }; title.onmouseout = function() { title.innerText = "👋 JavaScript 요원 다녀감 😆"; title.style.transform = ""; };

💡 부드러운 움직임 더하기

앗, 움직임이 너무 뚝뚝 끊기나요? 아래 줄에 트랜지션(transition)을 한 줄 추가해서 스르륵 부드럽게 만들어 주자!

힌트: style.transition 추가
title.style.transition = "all 0.5s ease";

✨ JS 작동 원리 탐구

· 기능 모음집 (function() { }): 여러 기능을 함께 실행할 때 쓰는 '기능 모음집'! 이벤트에 모음집을 달아주면, 이벤트 발생 시 기능들이 순서대로 실행되지!
· 부드러운 전환 (transition): 미리 전환 효과를 걸어두어, 이벤트가 일어날 때 영화처럼 스르륵 부드럽게 연출된단다!
· 원상 복구 ("" 빈 문자열): 마우스가 떠났을 때 원래대로 되돌리기 위해 빈 문자열("")을 대입하면, 추가했던 마법이 깔끔하게 지워진단다!

미션 8

사진 크기 조절하기 (마우스 이벤트 복습)

이번에는 방금 배운 마우스 이벤트를 사진에 적용해서 복습해 보자꾸나!

[1] 사진(아이디: main-img)을 찾아 변수에 저장하고
[2] 마우스가 닿을 때(onmouseover) 0.9배로 작아지게 만들고
[3] 마우스가 떠날 때(onmouseout) 원래 크기로 되돌려 보렴!

힌트: document.getElementById, onmouseover, onmouseout 사용
const img = document.getElementById('main-img'); img.onmouseover = function() { img.style.transform = "scale(0.9)"; }; img.onmouseout = function() { img.style.transform = ""; };

✨ JS 요원 아저씨와 CSS 요원의 꿀팁

이번엔 자바스크립트로 transition을 쓰지 않았는데도 부드럽게 움직였지? CSS 요원이 미리 사진(#main-img)에 전환 효과를 걸어두었기 때문이란다!
실무에서도 이렇게 뼈대(전환 효과)는 CSS에 맡기고, JS로는 크기나 위치만 바꾸는 방식을 자주 쓴단다.

미션 9

버튼에 알림 기능 모음집 연결하기

이번에는 환영 알림을 띄우는 나만의 모음집(function)을 만들고, 버튼과 연결해서 직접 발동시켜 보자!

[1] showMagic 이라는 이름의 모음집을 만들고 알림창(alert) 마법 넣기
[2] 마법 버튼(아이디: magic-btn)을 찾아 변수에 저장하기
[3] 버튼을 클릭할 때(onclick) showMagic 모음집이 실행되도록 연결하기

힌트: function, alert, document.getElementById, onclick 사용
function showMagic() { alert("환영합니다!"); } const btn = document.getElementById('magic-btn'); btn.onclick = showMagic;

💡 추가 미션: 다른 팝업창 띄워보기

· prompt("요원의 이름을 입력하세요"); : 텍스트를 입력받는 창
· confirm("정말 마법을 발동하시겠습니까?"); : 확인/취소를 고르는 창

✨ JS 작동 원리 탐구: 이름 있는 마법 vs 1회용 마법

· 이름 있는 마법 (function showMagic()): 마법에 이름을 쾅! 붙여두면 여러 버튼에서 똑같은 마법을 계속 돌려 쓸 수 있어요.
· 1회용 마법 (btn.onclick = function()): 미션 7, 8처럼 한 곳에서 딱 한 번만 쓸 마법이라면, 굳이 이름을 짓지 않고 이렇게 바로 연결하는 방식을 가장 많이 쓴단다!

⚠️ 특급 주의: 연결할 때 .onclick = showMagic();처럼 ()를 붙이면 바로 실행돼요!
버튼을 누를 때만 실행되게 하려면 괄호 없이 이름만 연결해야 해!

미션 10

비밀 통신망에 기록 남기기 (콘솔)

방금 배운 팝업창(prompt)을 활용해 투입할 요원의 이름을 입력받고, 비밀 통신망인 콘솔(console.log)과 화면의 제목을 동시에 바꿔보자!

[1] prompt 로 투입할 요원의 이름을 물어보고 변수에 저장하고
[2] console.log 를 이용해 콘솔에 투입 준비 완료 메시지를 남기고
[3] 미션 2에서 배웠던 title.innerText를 활용해 화면의 제목 내용도 함께 바꿔 보렴!

💡 비밀 통신망 확인법

키보드에서 F12(또는 Mac은 Cmd+Option+I)를 눌러 '개발자 도구'를 연 뒤, Console(콘솔) 탭을 확인해 보렴!

힌트: const, prompt, console.log 사용 (+ 기호로 텍스트 붙이기)
const agentName = prompt("투입할 요원의 이름을 알려주세요:"); console.log(agentName + " 요원, 투입 준비 완료!"); title.innerText = agentName + " 요원, 투입 준비 완료!";

💡 추가 미션: 마법 버튼에 연결하기

코드를 작성하자마자 바로 창이 뜨는 게 조금 아쉽지? 미션 9에서 배운 것처럼 이 코드를 새로운 모음집(function)으로 묶어서 마법 버튼(btn)을 누를 때만 작동하게 만들어 보렴!

btn.onclick = function() { // 아래 줄에 버튼을 누르면 실행할 코드를 넣기 };
