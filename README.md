# 🎡 도로랜드 스마트 가이드 (Doroland Smart Guide) Version 1.7.4 (2026.06.29.)

**"도로랜드 정보국 신입 요원 훈련 포털 및 스마트 가이드 실습 저장소입니다."**

이 프로젝트는 초중등 학생들을 대상으로 **공공데이터(기상청, 에어코리아) API**를 활용하여 실제 작동하는 웹 애플리케이션을 직접 만들어보는 훈련 코스를 제공합니다. 학생들은 뼈대 코드(Skeleton)를 바탕으로 미션을 해결하며 "공동 제작자(and You)"로서 프로젝트를 완성하게 됩니다.

## 📁 훈련 단계 및 프로젝트 구조

저장소는 총 3단계의 웹 특수 요원 훈련 과정으로 설계되어 있습니다.

### 1️⃣ `api-test/` | API 해킹 연습장 (Protocol Lab)
실제 공공데이터 서버와 통신하며 데이터를 해독(Parsing)하는 기초 훈련을 수행합니다.
*   **날씨 예보:** 기상청 초단기 강수 예측 API 연동 및 데이터 분석
*   **공기 질:** 에어코리아 실시간 대기오염 정보 API 연동 테스트

### 2️⃣ `agent-training/` | 웹 특수작전팀 훈련소 (Core Training)
HTML, CSS, JS 요원의 안내를 받아 웹의 3요소를 정복합니다.
*   **HTML 요원:** 문서의 뼈대와 구조 구축 (Semantic HTML)
*   **CSS 요원:** 세련된 디자인과 레이아웃 설계 (Modern CSS)
*   **JS 요원:** 생동감 넘치는 동적 로직 구현 (Vanilla JS)
*   **JS 요원 (API):** 강아지/고양이 오픈 API 통신(GET) 및 가상 서버(Mock API)를 활용한 방명록 CRUD(POST, DELETE) 심화 훈련

### 3️⃣ `smart-guide-web/` | 스마트 가이드 프로토타입 (Final Mission)
훈련을 통해 배운 기술을 집약하여 완성할 '도로랜드 스마트 안내 시스템'의 최종 모델입니다.
*   학생들은 메인 포털에서 **`smart-guide-web.zip`**을 다운로드하여 자신만의 안내 시스템을 구축합니다.
*   **대화형 에이전트 코딩 (Agentic Coding):** 동료 조력자인 AI 요원에게 대화로 요청을 하면, AI가 직접 코드를 작성하여 실제 기상청/에어코리아 API를 연동합니다. 학생들은 동료와 협업하며 프롬프트 엔지니어링 능력을 기릅니다.

## ✨ 주요 특징 (Premium Features)

도로랜드 스마트 가이드는 교육적 목적과 함께 최신 웹 기술 표준을 준수합니다.

*   **🎮 히든 스테이지 해금(Gamification Unlock):**
    *   JS 요원 기초 훈련 미션을 모두 완수해야만 심화 과정인 'API 훈련소' 스테이지가 포털과 메뉴에 잠금 해제(localStorage)되어 나타나도록 설계하여, 학생들의 학습 동기와 성취감을 극대화했습니다.
*   **🛠 자체 가상 서버(Mock API) 환경:**
    *   외부 백엔드 서버 구축 없이도 `fetch` 요청을 가로채어 실제 서버처럼 응답하는 내장 Mock API를 구현했습니다.
    *   브라우저 `localStorage`와 연동하여 단순 데이터 조회(GET)를 넘어 방명록 작성(POST), 삭제(DELETE) 등 실감 나는 통신 실습이 가능합니다.
*   **지능형 테마 시스템 (Smart Theme):**
    *   사용자 시스템 설정(Windows/Mac/iOS/Android 다크모드)을 자동 감지합니다.
    *   모든 페이지 간의 주/야간 모드 설정이 실시간으로 동기화(localStorage Persistence)됩니다.
*   **지능형 양방향 하이라이팅 (Bidirectional Inspector):**
    *   **코드 ➔ 화면:** 에디터에서 특정 라인에 마우스를 올리면 결과창의 해당 요소가 요원별 색상(HTML:주황, CSS:파랑, JS:노랑)으로 표시됩니다.
    *   **화면 ➔ 코드:** 결과창의 요소를 호버하면 이를 구성하는 HTML 뼈대, CSS 디자인, JS 로직 라인이 에디터에서 동시에 빛납니다.
    *   **변수 추적:** 단순 선택자뿐만 아니라 JS 변수에 할당된 조작 로직까지 지능적으로 추적하여 하이라이트합니다.
*   **표준화된 UI/UX:**
    *   **고가독성 폰트:** 가독성이 뛰어난 `Pretendard Variable` 폰트를 전 시스템에 적용했습니다.
    *   **통합 컨트롤러:** 모든 페이지 우측 상단에 포털 이동 및 테마 토글 버튼을 배치하여 일관된 내비게이션을 제공합니다.
    *   **커스텀 스크롤바:** 각 페이지의 테마와 아이덴티티에 맞춘 프리미엄 스크롤 디자인을 적용했습니다.
*   **참여자 존중 크레딧:** 
    *   최종 제작물인 **`smart-guide-web`** 폴더 내 파일 하단에는 코드를 완성한 학생들을 위한 공동 제작자 표기(`© 2026 XYLO & You.`)가 포함됩니다.
*   **✨ 스마트 코드위스퍼 (Smart Codewhisper):**
    *   **지능형 자동완성:** 단순 키워드 매칭을 넘어 HTML/CSS/JS의 문맥을 완벽히 인지하는 AI 스타일 힌트 엔진입니다.
    *   **프로젝트 인지형 스캔:** 현재 에디터에 정의된 모든 HTML 클래스/ID, JS 변수 및 함수명을 실시간으로 스캔하여 맞춤형 힌트를 제공합니다. (아이콘 구분: `⚡함수`, `📦변수`, `🎨클래스/ID`)
    *   **연쇄적 워크플로우:** 
        *   **JS:** `.style.color` 선택 시 자동으로 ` = '`를 완성하고 즉시 색상 목록을 띄워줍니다.
        *   **JS:** `getElementById` 등 DOM 메소드 선택 시 자동 괄호 삽입 및 문서 내 ID 목록을 제안합니다.
        *   **CSS:** 선택자 위치에서는 HTML 태그와 클래스를, 중괄호 안에서는 속성값과 콜론(: )을 자동으로 보정합니다.
    *   **교육용 백과사전 연동:** 모든 힌트에 한글 설명을 결합하여, 코드를 작성하며 동시에 문법의 의미를 학습할 수 있습니다.
*   **✨ 스마트 자동 들여쓰기 (Smart Auto-Indent):**
    *   **지능형 레이아웃 정렬:** 지령서의 다중행 코드를 복사하여 편집기에 붙여넣을 때, 현재 커서 위치의 들여쓰기 수준을 자동으로 분석하여 코드를 완벽하게 정렬합니다. 다중행 작업 시 일일이 들여쓰기를 수정해야 하는 번거로움을 자동화했습니다.
*   **💾 실시간 코드 세션 보존 및 내보내기 (Persistence & Export):**
    *   **자동 저장:** 에디터에 작성한 모든 코드는 브라우저에 실시간으로 안전하게 저장되어, 새로고침하거나 브라우저를 껐다 켜도 작업 내용이 유지됩니다.
    *   **코드 내보내기:** 작성한 코드를 단 한 번의 클릭으로 `.html` 파일로 다운로드하여 자신의 컴퓨터에 영구 소장할 수 있습니다.
    *   **코드 초기화:** 언제든 처음 상태로 되돌릴 수 있는 '초기화(Reset)' 버튼을 통해 반복 학습과 실험이 가능합니다.
*   **📸 글로벌 스크린샷 도구 (실험적 기능):**
    *   **실행 단축키:** `Cmd + Shift + S`
    *   **주요 기능:** 페이지 내 어떤 요소든 클릭 한 번으로 고화질(3x) 이미지로 캡처하여 클립보드에 복사합니다.
    -   **스마트 내비게이션:** `↑` (위 화살표) 키로 상위 부모 요소를 선택할 수 있으며, 실시간 요소 정보(태그/ID/클래스)를 툴팁으로 제공합니다.
    -   **통합 캡처:** 실시간 프리뷰 화면(Iframe) 내부 요소까지 완벽하게 지원하여 PPT 및 교육 자료 제작 효율을 극대화합니다.
*   **🤖 대화형 에이전트 코딩 (Agentic Coding) 탑재:**
    *   최종 미션(`smart-guide-web`)에는 코드를 모르는 학생들도 대화만으로 프로젝트를 완성할 수 있도록 든든한 동료 조력자 AI 지침(`AGENT.md`)이 탑재되어 있습니다. AI 요원은 학생의 지시에 따라 다양한 특수 모드로 완벽하게 작동합니다.
    *   **기본 통신 및 디버깅:** "가짜 데이터 지우고 진짜 날씨 가져와줘"라고 지시하면 실제 공공데이터 API 코드를 작성하며, 에러 발생 시 콘솔 로그를 분석하여 즉시 버그를 퇴치합니다.
    *   **문서 기반 확장 (RAG):** 학생이 새로운 오픈 API 가이드 문서(.pdf, .xlsx)를 폴더에 넣고 지시하면, AI가 스스로 문서를 읽고 새로운 데이터를 웹페이지에 추가해 줍니다.
    *   **테마 창작 및 고급 UI:** "우주 테마로 바꿔줘" 같은 요청에 맞춰 창의적인 디자인을 구현하며, 마우스 호버나 파티클 애니메이션 같은 고급 시각 효과도 적극적으로 제안합니다.
    *   **반응형 레이아웃 교정:** "스마트폰에서 화면이 깨져요"라고 피드백하면, 미디어 쿼리(`@media`)를 활용해 모바일 화면에 맞게 CSS 레이아웃을 즉시 수정해 줍니다.
    *   **데이터 시각화 모드:** "데이터를 그래프로 그려줘"라고 지시하면 Chart.js를 연동해 데이터 변화를 막대그래프나 꺾은선 차트로 그려내고 쉬운 비유로 원리를 설명합니다.
    *   **배포 가이드 모드:** "이거 어떻게 배포해?"라고 물으면 Netlify Drop을 활용한 직관적인 드래그 앤 드롭 무료 호스팅 방법과 주소 변경 팁을 친절하게 안내합니다.
