# 🎢 도로랜드 스마트 안내 시스템 (작전 지시서) Version 1.7.3 (2026.06.06.)

환영합니다, 특수작전팀 요원! 👋
여러분의 최종 임무는 도로랜드 방문객들을 위한 **'스마트 안내 시스템'**을 완성하는 것입니다.
현재 웹사이트의 뼈대(HTML)와 디자인(CSS)은 완벽하지만, 화면에 뜨는 날씨와 공기 질 데이터는 가짜(샘플 데이터)입니다. 여러분의 든든한 동료 조력자와 함께 이 웹사이트에 진짜 생명력을 불어넣어 주세요!

---

## 🎯 여러분의 최종 목표 (Mission Goal)
이제 여러분은 코드를 직접 타이핑하지 않습니다! 여러분에게는 여러분의 코딩을 대신해 줄 **AI 요원**이 대기하고 있습니다.
채팅창을 통해 동료 AI 요원에게 요청하여 가짜 데이터를 지우고, **실제 기상청과 에어코리아의 데이터**를 화면에 연결하도록 지시하세요!

## 🤖 강력한 'AI 요원' 소환 및 활용법
이 폴더(`smart-guide-web`)에는 여러분의 요청을 기다리는 동료 **AI 요원**이 대기하고 있습니다!

> ⚠️ **주의사항:** 훌륭한 요원은 동료에게 구체적이고 명확하게 요청해야 합니다! AI에게 다음과 같이 지시해 보세요.

*   **⚡ 기본 작전 지시:** *"요원! 지금 화면에 뜨는 가짜 날씨 데이터를 지우고, 기상청 API를 사용해서 진짜 날씨를 가져오도록 코드를 직접 수정해 줘!"*
*   **📍 좌표 변환 지시:** *"요원! `docs` 폴더에 있는 기상청 엑셀 파일을 읽고, 부산광역시 해운대구의 nx, ny 좌표를 찾아서 코드를 수정해 줘!"*
*   **🐛 에러 대처 요청:** 화면에 데이터가 안 뜰 때는 당황하지 마세요! *"요원! 화면에 데이터가 안 떠. 무슨 에러가 났는지 스스로 분석하고 코드를 고쳐줘!"* 라고 요청하세요.
*   **🎨 내 맘대로 꾸미기:** *"배경화면을 우주 테마로 바꾸고 싶어! CSS 코드를 수정해서 게임처럼 멋지게 만들어 줘"* 라고 요청해 보세요. AI가 즉시 코드를 고쳐줄 겁니다.

## 🚀 프로젝트 실행 방법 (작전 화면 띄우기)
AI 요원이 작전을 잘 수행했는지 확인하려면 어떻게 해야 할까요?
1. 내 컴퓨터 폴더에 있는 **`index.html` 파일을 마우스로 더블 클릭(빠르게 두 번 클릭)** 하세요.
2. 크롬(Chrome)이나 엣지(Edge) 같은 웹 브라우저가 열리면서 작전 화면이 나타납니다.
3. AI 요원이 코드를 수정했다는 보고를 하면, 브라우저에서 **새로고침(F5)**을 누르세요. 변경된 작전 화면을 바로 확인할 수 있습니다!

## 🏆 보너스 도전 과제 (Bonus Missions)
기본 미션을 완수했다면, 아래의 심화 작전에도 도전해 보세요!
1.  **나만의 테마 적용:** 배경 이미지, 버튼 색상, 글씨체를 내가 좋아하는 테마로 완전히 바꿔달라고 요청하기!
2.  **새로운 공공데이터 추가:** 공공데이터포털(`data.go.kr`)에서 맛집, 교통정보 등 새로운 API의 **가이드 문서(.pdf, .docx, .xlsx)**를 다운받아 `docs` 폴더에 넣고, AI 요원에게 새로운 기능을 추가해 달라고 문서와 함께 지시하기!
3.  **데이터 시각화(그래프) 도전:** AI 요원에게 *"가져온 날씨나 미세먼지 데이터를 예쁜 막대그래프나 꺾은선 차트로 그려줘!"* 라고 지시해서 나만의 데이터 대시보드 만들기!
4.  **내 작품 배포하기:** AI 요원에게 *"이거 내 친구들한테 링크로 보여주고 싶은데, 어떻게 무료로 배포해? 방법을 알려줘"* 라고 물어보고 전 세계에 내 웹사이트를 공개해 보기!
